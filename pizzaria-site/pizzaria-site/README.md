# Pizzaria Site

Bem-vindo ao projeto da Pizzaria Domidogs! Este repositório contém o código-fonte para um site de pizzaria interativo, onde os usuários podem visualizar o menu, adicionar itens ao carrinho e finalizar pedidos.

## Estrutura do Projeto

```
pizzaria-site
├── src
│   ├── app.js               # Ponto de entrada da aplicação
│   ├── index.html           # Estrutura HTML do site
│   ├── style.css            # Estilos CSS para o site
│   ├── components           # Componentes da aplicação
│   │   ├── menu.js          # Renderiza o menu de pizzas
│   │   ├── cart.js          # Gerencia o carrinho de compras
│   │   └── checkout.js      # Lida com o processo de finalização do pedido
│   └── data
│       └── pizzas.json      # Dados das pizzas disponíveis
├── package.json             # Configuração do npm
└── README.md                # Documentação do projeto
```

## Instalação

1. Clone o repositório:
   ```
   git clone <URL do repositório>
   ```
2. Navegue até o diretório do projeto:
   ```
   cd pizzaria-site
   ```
3. Instale as dependências:
   ```
   npm install
   ```

## Uso

Para iniciar o projeto, você pode abrir o arquivo `index.html` em um navegador. A aplicação permite que você:

- Visualize o menu de pizzas.
- Adicione pizzas ao carrinho.
- Finalize seu pedido com informações de pagamento.
