# Pizzaria Site

Este projeto é um site para uma pizzaria, desenvolvido com HTML, CSS e JavaScript. O site apresenta um menu de pizzas, permite que os usuários adicionem itens ao carrinho e realizem o checkout.

## Estrutura do Projeto

O projeto possui a seguinte estrutura de arquivos:

```
pizzaria-site
├── src
│   ├── index.html         # Página principal do site
│   ├── app.js             # Lógica JavaScript do site
│   ├── style
│   │   └── main.css       # Estilos CSS do site
│   └── assets
│       └── favicon.ico     # Ícone do site
├── package.json           # Configuração do npm
└── README.md              # Documentação do projeto
```

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Como Executar o Projeto

1. Clone o repositório:
   ```
   git clone <URL_DO_REPOSITORIO>
   ```

2. Navegue até o diretório do projeto:
   ```
   cd pizzaria-site
   ```

3. Instale as dependências (se houver):
   ```
   npm install
   ```

4. Abra o arquivo `src/index.html` em um navegador para visualizar o site.
